# Constitution — ApprovalFlow Lite (aprovação de solicitações internas)

## Principles
- Ship a working PoC fast.
- Prefer clarity over cleverness.
- Match the Figma layout for the main flows.
- Do not commit secrets.

## Non-goals / Out of scope
integração real com ERP

## Success criteria
A liderança (ou stakeholder) consegue testar o fluxo ponta a ponta em uma demo curta e concluir que a ideia é válida para seguir para uma próxima fase.
